## 💡 Что вам нужно сделать:

### Минимум (для работающего проекта):
1. ✅ Запустить frontend: `cd frontend && npm start`
2. ✅ Открыть http://localhost:3000

### Для красоты (опционально):
1. 📸 Добавить 10-20 картинок товаров
   - Через админку: http://localhost:8000/admin/
   - Или массово: `python manage.py add_images ../images_zoo --by-name`

### Для защиты:
1. 📖 Прочитать `course.txt`
2. 🎯 Подготовить демонстрацию
3. 💪 Уверенно защищаться!
