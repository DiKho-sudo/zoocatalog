## 📸 Изображения товаров - Краткая инструкция

### Где хранятся:
```
backend/media/products/ ← Физические файлы
PostgreSQL             ← Путь к файлу (image: "products/photo.jpg")
```

### Куда складывать:

**Способ 1: Через админку** (по одному)
1. http://localhost:8000/admin/products/product/
2. Найти товар → Загрузить → Сохранить

**Способ 2: Массово** (много товаров)
1. Создать папку `images_zoo` в корне
2. Положить туда JPG/PNG файлы
3. Назвать файлы ключевыми словами товара
4. Запустить:
   ```bash
   cd backend
   python manage.py add_images ../images_zoo --by-name
   ```

### Несколько фото для товара:
1. Откройте товар в админке
2. Прокрутите вниз до "Product images"
3. Добавьте дополнительные фото
4. Установите порядок (order: 1, 2, 3)

### URL в БД:
```sql
-- В PostgreSQL хранится путь:
image = "products/royal-canin_abc123.jpg"

-- Frontend формирует полный URL:
http://localhost:8000/media/products/royal-canin_abc123.jpg
```

Подробнее → **`РАБОТА_С_ИЗОБРАЖЕНИЯМИ.md`**

## 🐳 Запуск с Docker

```bash
# Установите Docker Desktop
# Затем одна команда:

docker-compose up

# Импорт данных:
docker cp stock_price.xlsx zoo_backend:/app/
docker-compose exec backend python fix_categories.py
docker-compose exec backend python manage.py import_simple stock_price.xlsx
docker-compose exec backend python manage.py auto_categorize

# Создать админа:
docker-compose exec backend python manage.py createsuperuser
```
