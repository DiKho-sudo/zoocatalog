## 2. 📸 Работа с изображениями

### Куда складывать изображения:

**Физически:**
```
backend/media/products/  ← Сюда Django сохраняет файлы
```

**Способы добавления:**

1. **Через админку** (для 1-10 фото):
   - http://localhost:8000/admin/products/product/
   - Найти товар → Поле "Image" → Выбрать файл → Сохранить

2. **Массово** (для 50+ фото):
   ```bash
   # Создать папку images_zoo, положить JPG/PNG
   # Назвать файлы ключевыми словами товара
   python manage.py add_images ../images_zoo --by-name
   ```

### Как они появляются на сайте:

1. **Django сохраняет файл** в `media/products/`
2. **Путь записывается в БД:** `"products/photo.jpg"`
3. **Frontend получает URL** через API: `/media/products/photo.jpg`
4. **Браузер загружает:** `http://localhost:8000/media/products/photo.jpg`

### URL в базе данных:

**PostgreSQL хранит:**
```sql
image = "products/royal-canin_abc123.jpg"  ← Только путь!
```

**НЕ хранится:** Само изображение (файл на диске)

### Несколько фотографий:

**Через админку:**
1. Откройте товар
2. Прокрутите вниз до секции **"Product images"**
3. Нажмите "Добавить еще одно изображение товара"
4. Загрузите фото
5. Установите порядок (order: 1, 2, 3...)
6. Сохраните

**Результат:**
- Главное фото в поле `image`
- Дополнительные в таблице `products_productimage`
- На странице товара появится галерея с переключением

Подробнее → **`РАБОТА_С_ИЗОБРАЖЕНИЯМИ.md`**

## 4. 📂 Распределение товаров по категориям

### НЕ вручную! Автоматически! 🤖

**Команда:**
```bash
python manage.py auto_categorize
```

**Результат:**
```
Товаров распределено по категориям: 1274
Товаров распределено по типам животных: 965
```

### Как работает:

Анализирует **название товара** и ищет ключевые слова:

```python
"Корм SIRIUS сухой для собак"
  → Категория: "Сухие корма"
  → Тип животного: "Собаки"

"Миска керамическая для кошек"
  → Категория: "Миски"
  → Тип животного: "Кошки"

"Лоток для туалета"
  → Категория: "Лотки"
```

### Правила категоризации:

Созданы в команде `auto_categorize.py`:
- Корма: "корм", "сухой", "консерв", "adult"
- Миски: "миска", "кормушка", "поилка"
- Туалеты: "туалет", "лоток", "наполнитель"
- Кошки: "кошк", "кот", "cat", "whiskas"
- Собаки: "собак", "dog", "pedigree"

### Если нужно вручную:

1. http://localhost:8000/admin/products/product/
2. Найти товар
3. Изменить поле "Category"
4. Сохранить

**Но это НЕ нужно!** Автоматизация уже сделала это за вас! 🎉
