# ⚡ Краткая справка - Что и как

## 🖼️ Изображения

### Куда складывать:
1. **Через админку:** http://localhost:8000/admin/products/product/
   - Найти товар → Загрузить → Сохранить
   
2. **Массово:** Создать папку `images_zoo`, положить JPG/PNG файлы
   ```bash
   python manage.py add_images ../images_zoo --by-name
   ```

### Где хранятся:
- **Физически:** `backend/media/products/`
- **В БД:** Путь к файлу (`products/photo.jpg`)
- **На сайте:** `http://localhost:8000/media/products/photo.jpg`

### Несколько фото:
- Админка → Товар → Секция "Product images" → Добавить

## 🔄 Обновление из 1С

```bash

```

Запускайте хоть каждый день!

## 🐳 Docker

```bash
# Запустить всё:
docker-compose up

# Остановить:
docker-compose down

# Импорт в Docker:
docker cp stock_price.xlsx zoo_backend:/app/
docker-compose exec backend python manage.py import_simple stock_price.xlsx
```
