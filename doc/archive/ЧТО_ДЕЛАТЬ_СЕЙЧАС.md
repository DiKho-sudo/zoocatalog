# ✅ Что делать прямо сейчас - Чеклист

## 🎯 Цель: Настроить PostgreSQL и импортировать товары из 1С

---

## Шаг 1: Создайте базу данных (2 минуты)

### В pgAdmin:
1. ☐ Откройте **pgAdmin 4**
2. ☐ Подключитесь к серверу (введите пароль)
3. ☐ Правой кнопкой на **"Databases"** → **Create** → **Database**
4. ☐ Имя базы: **`zoo_catalog`**
5. ☐ Нажмите **Save**

---

## Шаг 2: Создайте файл .env (1 минута)

### В папке backend/:
1. ☐ Создайте **новый файл** с именем `.env` (точка в начале!)
2. ☐ Скопируйте содержимое из файла `env_example.txt`
3. ☐ Измените строку `DB_PASSWORD=ВАШ_ПАРОЛЬ_ОТ_POSTGRES`
4. ☐ Вставьте ваш реальный пароль от PostgreSQL

**Пример .env файла:**
```
DB_ENGINE=django.db.backends.postgresql
DB_NAME=zoo_catalog
DB_USER=postgres
DB_PASSWORD=mypassword123
DB_HOST=localhost
DB_PORT=5432
```

---

## Шаг 3: Установите драйвер (30 сек)

### В терминале:
```bash
cd backend
pip install psycopg2-binary
```

---

## Шаг 4: Примените миграции (1 минута)

```bash
python manage.py migrate
```

Должно быть:
```
✓ Applying products.0001_initial... OK
✓ Applying products.0002_product_unit... OK
```

---

## Шаг 5: Создайте админа (30 сек)

```bash
python manage.py createsuperuser
```

Введите:
- Username: **admin**
- Email: **(пропустите - Enter)**
- Password: **admin123**

---

## Шаг 6: Создайте категории (10 сек)

```bash
python fix_categories.py
```

Результат:
```
Всего создано: 26 категорий
Типов животных: 7
```

---

## Шаг 7: Импортируйте товары (1 минута)

```bash
python manage.py import_simple ..\stock_price.xlsx
```

Результат:
```
Создано товаров: 1874
```

---

## Шаг 8: Запустите всё! (30 сек)

### Backend:
```bash
python manage.py runserver
```

### Frontend (в другом терминале):
```bash
cd frontend
npm start
```

## 📝 Если что-то не работает:

### PostgreSQL не подключается:
1. Проверьте что PostgreSQL запущен
2. Проверьте пароль в `.env`
3. Проверьте что база `zoo_catalog` создана в pgAdmin

### Товары не импортируются:
1. Проверьте что файл `stock_price.xlsx` в папке `courseProj/`
2. Запустите: `python manage.py import_simple ..\stock_price.xlsx --clear`

### Frontend не показывает товары:
1. Проверьте что backend запущен (http://localhost:8000/api/products/)
2. Обновите страницу (Ctrl+F5)
