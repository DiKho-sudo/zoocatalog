# ⚡ Краткая инструкция - Импорт из 1С

## 🎯 У вас есть файл `про.xlsx` из 1С Розница

### Шаг 1: Импорт товаров (1 минута)

```bash
cd backend
python manage.py import_from_1c "D:\courseProj\про.xlsx"
```

**Результат:** Все товары из файла добавлены в каталог! ✅

---

### Шаг 2: Добавление цен

#### Вариант А: Через админку (быстро для 10-20 товаров)
1. http://localhost:8000/admin/products/product/
2. Открываете товар → Вводите цену → Сохраняете

#### Вариант Б: Из Excel файла (для большого количества)

**Создайте файл `цены.xlsx`:**

| Наименование | Цена |
|-------------|------|
| ECOLINE ROMEO GREEN туалет-домик 57x39x41 | 45.90 |
| Совок для корма средний бежевый 24 см | 3.50 |

**Запустите:**
```bash
python manage.py update_prices цены.xlsx
```

---

### Шаг 3: Добавление картинок

1. **Создайте папку:** `images_zoo`

2. **Положите картинки** с названиями товаров:
   ```
   ECOLINE.jpg
   Совок.jpg
   Лоток.png
   ```

3. **Запустите:**
   ```bash
   python manage.py add_images ../images_zoo --by-name
   ```

**Готово!** Все картинки добавлены! 🎉

---

## 📋 Полная последовательность:

```bash
# 1. Импорт товаров
python manage.py import_from_1c "про.xlsx"

# 2. Обновление цен (если есть файл с ценами)
python manage.py update_prices "цены.xlsx"

# 3. Добавление картинок
python manage.py add_images "../images_zoo" --by-name

# 4. Проверка
# Админка: http://localhost:8000/admin/
# Сайт: http://localhost:3000/catalog
```

---

## 🔄 Для регулярных обновлений:

```bash
# Обновить товары и цены из 1С
python manage.py import_from_1c "про.xlsx" --update

# Обновить только цены
python manage.py update_prices "цены.xlsx"
```

---

## 📸 Где брать картинки?

1. **Google Images**: скопируйте название товара → найдите фото
2. **Yandex Images**: то же самое
3. **Сайты производителей**: Зооэкспресс, TRIXIE, GAMMA
4. **У поставщика**: запросите архив с фотографиями

Назовите файлы по товару и команда `add_images` сама всё найдет!

---

## ❓ Проблемы?

**"Товар не найден"** → Проверьте название файла или используйте флаг `--by-name`  
**"Колонка не найдена"** → Проверьте названия колонок в Excel  
**"Цена не обновляется"** → Убедитесь что название товара совпадает точно
