# 🎯 Инструкция для вашего файла из 1С Розница

## Ваша выгрузка

Вижу в вашем файле `про.xlsx`:
- ✅ Артикул (пустой)
- ✅ Наименование (есть)
- ✅ Марка (Бренд) - Зооэкспресс, TRIXIE, GAMMA
- ✅ Наименование полное (можно использовать как описание)
- ❌ Цена (нет)
- ❌ Остаток (нет)

**Не проблема!** Команда импорта адаптирована под это.

---

## Шаг 1: Импорт товаров из вашего файла

```bash
cd backend

# Импортируйте ваш файл
python manage.py import_from_1c "D:\courseProj\про.xlsx"
```

**Что произойдет:**
- ✅ Создадутся товары из вашего файла
- ✅ Автоматически создадутся бренды (Зооэкспресс, TRIXIE, GAMMA и т.д.)
- ✅ Категория будет "Розничный товар" (из колонки "Вид номенклатуры")
- ⚠️ Цена будет 0 BYN (можно обновить позже через админку)
- ✅ "Наименование полное" станет описанием товара

---

## Шаг 2: Добавление цен (после импорта)

### Вариант А: Через админ-панель (для небольшого количества)

1. Откройте http://localhost:8000/admin/products/product/
2. Найдите товар
3. Введите цену
4. Сохраните

### Вариант Б: Выгрузите из 1С файл с ценами

1. В 1С сделайте выгрузку с колонками:
   - Наименование
   - Цена
2. Я создам скрипт для обновления цен по названию

### Вариант В: Вручную создайте Excel с ценами

Создайте файл `цены.xlsx`:

| Наименование | Цена |
|-------------|------|
| ECOLINE ROMEO GREEN туалет-домик 57x39x41 | 45.90 |
| Совок для корма | 3.50 |
| Лоток для кошек | 12.80 |

Я создам команду для импорта цен.

---

## Шаг 3: Добавление изображений

### Самый простой способ:

1. **Создайте папку** `images_zoo`

2. **Положите туда картинки** и назовите их по товару:
   ```
   images_zoo/
   ├── ECOLINE ROMEO GREEN.jpg
   ├── Совок для корма.jpg
   ├── Лоток.png
   ├── Туалет MEGAMIC.jpg
   └── ...
   ```

3. **Запустите команду:**
   ```bash
   python manage.py add_images ../images_zoo --by-name
   ```

**Готово!** Все картинки автоматически привязались к товарам! 🎉

## Полный процесс для вас:

```bash
# 1. Импортируйте товары
cd backend
python manage.py import_from_1c "D:\courseProj\про.xlsx"

# Вывод:
# Создано новых товаров: ~25 (сколько в вашем файле)

# 2. Добавьте цены через админку (пока вручную)
# Откройте http://localhost:8000/admin

# 3. Подготовьте картинки в папке images_zoo

# 4. Добавьте изображения
python manage.py add_images ../images_zoo --by-name

# Вывод:
# ✅ Добавлено изображений: 20
# ❌ Товары не найдены: 5 (для них нужно переименовать файлы)
```

## Где брать изображения?

### Вариант 1: Google/Yandex поиск
Для каждого товара:
1. Скопируйте название (например "ECOLINE ROMEO GREEN")
2. Вставьте в Google Images
3. Найдите подходящее фото
4. Сохраните как "ECOLINE ROMEO GREEN.jpg"

### Вариант 2: С сайтов производителей
- Зооэкспресс: https://zoo-express.by/
- TRIXIE: официальный сайт
- GAMMA: поиск в Google

### Вариант 3: Попросить у поставщика
Если есть контакты - запросите архив с фото товаров.

## Что делать с пустыми артикулами?

Ничего страшного! Система создаст артикул автоматически из названия:
- "ECOLINE ROMEO GREEN..." → slug: `ecoline-romeo-green-...`
- "Совок для корма" → slug: `covok-dlja-korma`
