# 🚀 Быстрый старт: Импорт из 1С за 5 минут

## Шаг 1: Выгрузите товары из 1С (2 минуты)

1. Откройте **1С: Розница 8.3**
2. **Товары** → **Номенклатура**
3. **Еще** → **Вывести список**
4. **Файл** → **Сохранить как Excel**
5. Сохраните как `товары.xlsx`

Убедитесь что есть колонки: **Наименование**, **Артикул**, **Цена**

## Шаг 2: Подготовьте изображения (опционально)

Создайте папку `images_1c` и положите туда картинки:
- Имя файла = Артикул товара
- Например: `RC001.jpg`, `WH002.png`

## Шаг 3: Запустите импорт (1 минута)

Откройте терминал в папке проекта:

```bash
cd backend

# Установите библиотеку для Excel (первый раз)
pip install openpyxl

# Запустите импорт
python manage.py import_from_1c путь\к\товары.xlsx
```

**С изображениями:**
```bash
python manage.py import_from_1c товары.xlsx --images-dir images_1c
```

## Готово! ✅

Проверьте результат:
- Админка: http://localhost:8000/admin/products/product/
- Сайт: http://localhost:3000/catalog

---

## Для обновления (следующие разы):

```bash
python manage.py import_from_1c товары.xlsx --update
```

Флаг `--update` обновит цены и остатки существующих товаров.

---

## Если что-то не работает:

1. **"Файл не найден"** → Укажите полный путь: `D:\товары.xlsx`
2. **"Колонка не найдена"** → Переименуйте колонки в Excel на русские
3. **"Изображения не загружаются"** → Проверьте имена файлов = артикулам
