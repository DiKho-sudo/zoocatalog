# 🐾 Веб-каталог зоомагазина

> Курсовой проект БГУ ФПМИ, 2025

Динамический веб-каталог зоомагазина с интеграцией 1С Розница, системой поиска и фильтрации товаров.

## ✨ Особенности проекта

- 📦 **1874 товара** из реальной учетной системы 1С Розница
- 🔄 **Автоматическая синхронизация** цен и остатков
- 🎨 **Современный UI** с мягкой зеленой гаммой и анимациями
- 💡 **Умный поиск** с автоподсказками и debounce
- 📂 **27 категорий** товаров с иерархической структурой
- 🐾 **6 типов животных** для удобной навигации
- 📱 **Адаптивный дизайн** для всех устройств
- 🐘 **PostgreSQL** для надежного хранения данных
- 🐳 **Docker** для простого развертывания

## 🚀 Быстрый старт

### Вариант 1: С Docker (рекомендуется)

```bash
docker-compose up
```

Откройте http://localhost:3000

### Вариант 2: Локально

**Backend:**

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

**Frontend:**

```bash
cd frontend
npm install
npm start
```

## 📊 Импорт данных из 1С

```bash
cd backend

# Создать категории
python fix_categories.py

# Импортировать товары
python manage.py import_simple ..\stock_price.xlsx

# Автоматически распределить по категориям
python manage.py auto_categorize
```

## 🎯 Функциональность

### Каталог товаров

- Сетка товаров с изображениями и ценами
- Информация о производителе, весе, единицах измерения
- Бейджи статуса наличия и специальных характеристик
- Пагинация (12 товаров на страницу)

### Поиск и фильтрация

- Полнотекстовый поиск с автоподсказками
- Фильтрация по:
  - Ценовому диапазону
  - Категории товара
  - Типу животного
  - Производителю
  - Возрастной группе
  - Специальным характеристикам
- Компактная пагинация с навигацией

### Интеграция с 1С

- Импорт номенклатуры
- Синхронизация цен и остатков
- Автоматическое определение статуса наличия
- Автоматическая категоризация товаров

### Административная панель

- Управление товарами, категориями, брендами
- Загрузка изображений
- Добавление нескольких фото к товару
- Фильтры и поиск
