# Веб-каталог зоомагазина - Обзор проекта

## ✅ Что реализовано

### Backend (Django)
- ✅ Структура Django проекта с приложением `products`
- ✅ 5 моделей базы данных:
  - `Product` - товары с полной информацией
  - `Category` - иерархические категории
  - `Brand` - производители
  - `AnimalType` - типы животных
  - `ProductImage` - дополнительные изображения
- ✅ REST API с использованием Django REST Framework
- ✅ Система фильтрации (цена, бренд, категория, тип животного и т.д.)
- ✅ Полнотекстовый поиск
- ✅ Пагинация (12 товаров на странице)
- ✅ Сортировка результатов
- ✅ Админ-панель для управления каталогом
- ✅ CORS настройки для работы с React
- ✅ Оптимизированные запросы к БД (select_related, prefetch_related)
- ✅ Индексы для ускорения поиска

### Frontend (React)
- ✅ 3 основные страницы:
  - Главная страница с категориями и новинками
  - Страница каталога с фильтрами
  - Детальная страница товара
- ✅ Компоненты:
  - Header с поиском и навигацией
  - ProductCard - карточка товара
  - Filters - панель фильтров
- ✅ React Router для навигации
- ✅ Axios для API запросов
- ✅ Адаптивный дизайн (мобильные, планшеты, десктоп)
- ✅ Интерактивная фильтрация без перезагрузки
- ✅ Галерея изображений на странице товара
- ✅ Хлебные крошки для навигации
- ✅ Пагинация списка товаров

### Документация
- ✅ Полный текст курсовой работы (`course.txt`)
- ✅ README с описанием проекта
- ✅ SETUP.md с пошаговыми инструкциями
- ✅ Комментарии в коде

## 🎯 Ключевые возможности

1. **Просмотр каталога**
   - Сетка товаров с изображениями и ценами
   - Информация о производителе и весе
   - Бейджи специальных характеристик

2. **Поиск**
   - Поиск в шапке сайта
   - Поиск по названию, описанию и составу
   - Мгновенные результаты

3. **Фильтрация**
   - По ценовому диапазону
   - По производителю
   - По категории
   - По типу животного
   - По возрастной группе
   - Гипоаллергенные товары
   - Беззерновые товары
   - Комбинирование фильтров

4. **Детальная информация**
   - Полное описание товара
   - Состав продукта
   - Характеристики
   - Галерея изображений
   - Ссылка на сайт производителя

5. **Административная панель**
   - Управление товарами
   - Управление категориями
   - Управление брендами
   - Inline добавление изображений
   - Фильтры и поиск в админке

## 🚀 Технологии

### Backend
- **Django 6.0** - веб-фреймворк
- **Django REST Framework 3.16** - создание API
- **Django CORS Headers** - поддержка CORS
- **Django Filter** - фильтрация данных
- **Pillow** - работа с изображениями
- **Python 3.13**

### Frontend
- **React 18** - UI библиотека
- **React Router DOM 6** - маршрутизация
- **Axios** - HTTP клиент
- **CSS3** - стилизация

### База данных
- **SQLite** (для разработки)
- **PostgreSQL** (рекомендуется для продакшн)
