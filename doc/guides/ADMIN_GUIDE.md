# Руководство администратора Django Admin

## 🎯 Что исправлено

### ✅ 1. Редактирование страны производителя

Теперь при редактировании товара вы можете:
- Видеть полную информацию о производителе
- Изменять производителя через удобный поиск (autocomplete)
- В списке товаров видна колонка "Страна"

**Как использовать:**
1. Откройте товар для редактирования
2. В разделе "Классификация" → поле "Производитель"
3. Начните вводить название производителя
4. Выберите из выпадающего списка
5. Чтобы изменить страну производителя - нажмите на иконку "+" рядом с полем или откройте карточку производителя

---

### ✅ 2. Загрузка изображений товаров

**Типы изображений:**

#### A. Основное изображение товара
- Поле "Изображение" в разделе "Основная информация"
- Отображается в карточке товара
- Один файл на товар

#### B. Дополнительные изображения (галерея)
- Секция "Дополнительные изображения" внизу страницы
- Можно добавить неограниченное количество
- Поля:
  - **Изображение** - файл изображения
  - **Альтернативный текст** - описание (для SEO)
  - **Порядок** - сортировка (меньшее число = выше)

---

## 📁 Структура загрузки файлов

Изображения сохраняются в директории `media/`:

```
media/
├── products/          # Основные изображения товаров
├── brands/            # Логотипы производителей
├── categories/        # Изображения категорий
├── animal_types/      # Иконки типов животных
└── products/gallery/  # Дополнительные фото товаров
```

---

## 🖼️ Рекомендации по изображениям

### Форматы файлов:
- ✅ JPG/JPEG - для фотографий
- ✅ PNG - для изображений с прозрачностью
- ✅ WebP - для современных браузеров

### Размеры:
- **Основное фото товара**: 800x800px или 1200x1200px
- **Галерея**: 1200x1200px
- **Логотип производителя**: 300x300px
- **Иконка категории**: 200x200px

### Оптимизация:
- Сжимайте изображения перед загрузкой
- Рекомендуемый размер файла: до 500KB
- Для больших каталогов используйте WebP формат

## 📝 Работа с производителями

### Создание производителя:

1. Перейдите в **Производители** → **Добавить производителя**
2. Заполните поля:
   - **Название** - обязательное поле
   - **Страна** - страна производства
   - **Описание** - информация о бренде
   - **Логотип** - загрузите логотип (необязательно)
   - **Веб-сайт** - URL сайта производителя

3. Нажмите **Сохранить**

### Редактирование производителя:

1. Найдите производителя в списке
2. Нажмите на название
3. Измените нужные поля (включая **Страну**)
4. Сохраните изменения

## 🔧 Работа с товарами

### Добавление нового товара:

1. **Основная информация**:
   - Название
   - Slug (генерируется автоматически)
   - Описание
   - **Основное изображение** ← Загрузите фото здесь

2. **Характеристики**:
   - Состав
   - Вес/Количество + Единица измерения
   - Цена
   - Возрастная группа

3. **Классификация**:
   - Категория
   - **Производитель** ← Выберите или создайте производителя
   - Тип животного

4. **Особые свойства**:
   - Гипоаллергенный (✓/✗)
   - Беззерновой (✓/✗)
   - Статус наличия

5. **Дополнительные изображения** (внизу страницы):
   - Нажмите "+ Добавить еще один(у) Изображение товара"
   - Выберите файл
   - Укажите альтернативный текст
   - Установите порядок
   - Можно добавить несколько изображений

## ⚠️ Важные моменты

### 1. Проблемы с загрузкой изображений?

**Если изображение не загружается:**
- Проверьте размер файла (не больше 10MB)
- Проверьте формат (JPG, PNG, WebP)
- Проверьте права доступа к директории `media/`

**Для Docker:**
```bash
docker exec zoo_backend chmod -R 777 /app/media
```

### 2. Изменение страны производителя

**Способ 1: Через карточку производителя**
1. Производители → Найти производителя
2. Изменить поле "Страна"
3. Сохранить

**Способ 2: При редактировании товара**
1. Открыть товар
2. Нажать иконку "+" рядом с полем "Производитель"
3. Создать нового производителя с нужной страной
4. Или нажать иконку "редактировать" (карандаш) чтобы изменить существующего

### 3. Массовое обновление

Для массового изменения используйте:
- **Команды управления** (см. `DATA_INTEGRITY.md`)
- **Действия в админке** (выберите несколько товаров → действие)

## 🚀 Быстрый старт

### Добавление товара с фото:

1. **Перейдите**: Товары → Добавить товар
2. **Заполните основную информацию** и загрузите фото
3. **Выберите производителя** (autocomplete)
4. **Добавьте дополнительные фото** (если нужно)
5. **Сохраните**

### Изменение страны производителя:

1. **Перейдите**: Производители
2. **Найдите нужного** производителя
3. **Откройте карточку** и измените "Страна"
4. **Сохраните**

---

## 📞 Поддержка

При возникновении проблем:
1. Проверьте логи: `docker logs zoo_backend`
2. Проверьте права на media: `docker exec zoo_backend ls -la /app/media`
3. Перезапустите контейнер: `docker-compose restart backend`
